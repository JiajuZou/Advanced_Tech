## Google搜索语法

| 功能   | 语法 |     示例 |
| :----- | :--: | -------: |
| 限定关键词 |  英文引号""  |  |
| 限定标题 |  intitle:  |  |
| 限定标题多个关键词 |  intitle:"XX""YY"  |  |
| 限定内容关键词 | intext: | intext:"XiaoMing""SUSTech"|
| 限定网址关键词 | inrul: | |
| 限定网址来源 | site: | 2024推荐面试 site:Zheda.com|
| 限定图片大小 | imagesize: | SUSTech iamgesize:2560x1440
| 限定文件类型 | filetype: | XX研究报告 filetype:pdf | 
>以上搜索语法可相互组合
